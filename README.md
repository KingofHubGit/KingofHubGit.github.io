# Lucas 的个人博客

这里是我的博客仓库，用于记录技术实践、开发笔记和日常思考。

## 博客地址

- GitHub Pages: <https://kingofhubgit.github.io>

## 内容说明

- `/_posts`：已发布的博客文章
- `/_drafts`：草稿文章
- `/pages`：独立页面（如关于、归档等）
- `/images`：文章配图与站点资源
- `/_config.yml`：博客全局配置

## 本地预览

如果需要在本地预览，可使用 Jekyll：

```bash
bundle install
bundle exec jekyll serve
```

启动后访问：

- <http://127.0.0.1:4000>

## 写作与发布流程

1. 在 `/_posts` 新建文章，文件名格式：`YYYY-MM-DD-title.md`
2. 使用 Front Matter 配置标题、分类、标签等信息
3. 本地预览确认格式、链接和图片显示正常
4. 提交到 `main` 分支后由 GitHub Pages 自动构建发布

## 联系方式

如果你对文章内容有建议，欢迎通过 GitHub Issues 交流。
